# communications/

Donor updates, stakeholder briefings, and external communications materials.

## Naming convention

```
donor-update-YYYY-vN.html           ← Annual/periodic donor updates
stakeholder-brief-TOPIC-vN.html     ← Specific topic briefs
impact-report-YYYY-vN.html          ← Impact summaries
```

## Sensitivity reminder

- Do not include individual donor names without permission
- Aggregate impact figures are fine; individual contribution amounts are not
- WhatsApp-shared materials should be mobile-friendly (test at 375px width)
