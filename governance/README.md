# governance/

Board packs, meeting agendas, governance summaries, and organisational documents for TAHA board members and auditors.

## Naming convention

```
board-pack-YYYY-QN-vN.html          ← quarterly board packs
annual-report-YYYY-vN.html          ← annual reports
governance-summary-vN.html          ← standing governance overview
```

## Sensitivity reminder

Board packs may contain financial summaries. Before publishing:
- Remove individual donor names and amounts
- Use aggregate figures only
- Do not include audit management letters or full financial statements — link to secure shared drive instead
