# partnerships/

Visual summaries of TAHA's institutional partnerships — MoU structures, partner briefs, and collaboration frameworks.

## Important

This folder contains **visual summaries only** — not full legal documents.  
Full MoU text, SLA drafts, and signed agreements must be stored in a secure shared drive, not in this public repository.

## Naming convention

```
mou-summary-PARTNER-vN.html         ← MoU one-pagers
partner-brief-PARTNER-vN.html       ← partner overview cards
sla-summary-PARTNER-vN.html         ← SLA structure summaries
```
